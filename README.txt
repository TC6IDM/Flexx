to get this project, clone the repository, go to eclipse, file > new > Java project, Make sure "Create module-info.java file" option is OFF, browse to the inner folder (Flexx\Flexx), press Open, press finish. then to run the program, run Flexx.java


change the password in JDBC.java to your password for your sql


before running each new iteration, DROP DATABASE flexx; to insure any changes go through
